== 时间安排

历年统考时间都是12月中旬左右，假设复习时间从本年(2018年)11月1日开始，那么总共可以复习的时间为12个月。

从大纲可以看出，除复试外，各科目相互独立，没有直接关联，因此时间上可以并行安排。

复试依赖于初试各科目的一些基础知识，可以排到最后三个月同时准备。

=== 201英语一

英语一比较特殊，重点难点在于：

1. 单词
2. 语法
3. 阅读
4. 写作
5. 口语

.复习思路
1. 单词是整个英语一的基础，因此初期应当将重心放在单词上。
2. 当词汇量积累到一定程度之后，可以语法、阅读、口语并进，使之反哺单词。
3. 写作可以中后期开始练习和复习。
