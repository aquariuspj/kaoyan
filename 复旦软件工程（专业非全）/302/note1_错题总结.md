## 求解题

### 利用倒数进行通分

**【题目】**：求 $y = \ln( x + \sqrt{1 + x^2} )$ 的反函数  
**【解】**

1. 由原等式得 $e^y = x + \sqrt{1 + x^2}$
2. 求其倒数得

    $$e^{-y} = \frac{1}{x + \sqrt{1 + x^2}}$$

3. 又因为 $1 = (x + \sqrt{1 + x^2})(\sqrt{1 + x^2} - x)$ 得

    $$e^{-y} = \sqrt{1 + x^2} - x$$

4. 1式和3式相减再化简即可得到结果：

    $$x = \frac{e^y - e^{-y}}{2}$$
